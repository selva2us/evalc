System Prompts

Every Anthropic system prompt used across the Architect tool, AI Pipeline, and Asset Studio lives in prompts/*.txt and loads from here — nothing is hardcoded in the services/routes/pipeline files. Edits take effect on the next generation call, no restart needed. Dynamic prompts use $identifier placeholders (e.g. $technology_name) rather than curly braces, since several prompts instruct the model to respond in a literal JSON { } shape.
